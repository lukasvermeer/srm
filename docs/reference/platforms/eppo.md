#  Eppo

Default SRM Checks
{: .label .label-green }

[Eppo](https://www.geteppo.com) automatically warns users about Sample Ratio Mismatch. Documentation is available [here](https://docs.geteppo.com/statistics/sample-ratio-mismatch).
